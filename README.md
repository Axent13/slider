# Запуск проекта

Клонируем репозиторий:
`git clone https://github.com/Axent13/git.git`

Переходим в папку с проектом:
`cd slider/`

Устанавливаем зависимости:
`npm install`

Запускаем проект:
`npm run dev`

Проект будет доступен по адресу `http://localhost:8080/`
